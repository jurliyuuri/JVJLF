度量衡などについて考察する

以下のは雑記・考察ログです。すでに絶滅した過去の仕様などが数多く含まれている可能性があります。

1. 既存の設定

2017年7月7日〜7月8日辺りで、「ファイクレオネと現世の間の単位の対応」というところで、既に「時間」「音高」「温度」については決めている。

対応と定義を掲載しておくと、

  • 0.809259秒 <= 1 stususn <= 0.809267秒
  • 基準音高（「音楽について考察する」で定義された「一」）は243 stu^(-1)または244.3 stu^(-1)である。244.3とかいう妙な値は、何らかの権威によるものとされている。
    • 244.3のとき: 301.878[Hz] < 基準音高 < 301.882[Hz]
    • 243のとき: 300.271[Hz] < 基準音高 < 300.275[Hz]
  • la artidzemirs-kysonium（貴度）は水の凝固点と基準温度とを100等分した温度目盛りである。基準温度をzとすると、37.60134°C <= z <= 37.60185°C である。ということで1ak°≒摂氏0.37601度であり、摂氏温度に2.6594を掛ける事で求められることになる。
